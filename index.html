What is Web Accessibility?

"The power of the Web is in its universality.
Access by everyone regardless of disability is an essential aspect."
Tim Berners-Lee, W3C Director and inventor of the World Wide Web

Types of disability

  • Permanent Disability
  • Temporary Disability
  • Conditional Disability

Why this is important?

  • The web removes barriers of communication and interaction of the real world
  • Equal access
  • Ease of access to information

Important for Businesses

  • Extend Market Reach
  • Returning customers
  • SEO

Keyboard navigation

Keyboard-only users and users of assistive technology use the keyboard to navigate all website content and access every interactive element.

  • Check that all interactive elements receive tab order
  • Test all interactive elements for keyboard-focus activation
